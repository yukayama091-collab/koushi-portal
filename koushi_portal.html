ヘルプ・凡例

マーク・色・データソース・各ページの使い方

このポータルは

講師が担当している生徒の 進捗・卒業状況・講義スケジュール をひと目で把握するための画面です。 生徒DB（投稿プログラム／生徒情報／要サポート生徒分析）と講師DBの生徒管理シートを GAS Web App 経由で読み取り、ブラウザに表示しています。 数値はキャッシュされ、右上「 更新」で再読み込みできます。

アラートマーク（生徒の状態）
${''}
停滞中
最終ステップから 15日以上 進捗無し。早急に連絡を取る対象。
つまずき
最終ステップから 3〜14日 進捗無し。声かけのタイミング。
進行中
SP受講開始済みで、現在ステップを進めている状態。
卒業
プログラム完了（PP10完了）または 要サポート生徒分析 の 卒業生 フラグ＋卒業日が過去。
完了/順調
進捗に問題なし、または50日以内にプログラム完了。
SP未開始
受講開始日が未入力。名簿のみの生徒もここに含む。
ステータス Pill（色付きラベル）
50日達成SP→AP→PP を 50日以内 に完了
80日達成50〜80日で完了
80日超過80日を超えて完了
進行中受講開始から50日未満で未完了
期限切れ50日50〜80日経過しても未完了
期限切れ80日+80日を超えても未完了
SP未開始受講開始日が未入力
投稿プログラム ステップ方眼の色
SP完了（緑） — 【SP】Step1〜10 の完了日が入っている
SP未完（淡緑）
AP完了（オレンジ）
AP未完（淡オレンジ）
PP完了（青）
PP未完（淡青）

新カリキュラム（STEP1〜19）は青色1色のみで表示します。

ダッシュボード KPI の意味
進行中の生徒SP受講開始済みで、まだ完了していない生徒数（卒業生は除く）
50日達成率50日以内に完了した生徒 ÷ SP受講開始済みの生徒
卒業済みの生徒プログラム完了済 ＋ 卒業日が記録されている生徒の合計
要フォロー（停滞）最終ステップから15日以上止まっている生徒
1万アカ達成率フォロワー10,000人以上の生徒 ÷ フォロワーデータが取れる生徒
卒業予定日の計算ルール
  1. 基本: 第1回講義（初回HR） + 180日
  2. 例外: 180日経過時点で第12回講義がまだ実施されていない場合は、第12回講義の実施日が卒業日
  3. データソース: 講師の 生徒管理シート から第1回・第12回の実施日を取得（講師選択時に自動取得）
  4. フォールバック: 生徒管理シート未読込のクラスは 要サポート生徒分析 の 初回HR日 + 180日
各データの取得元
表示している情報取得元シート取得列
進捗 / 開始日 / ステータス生徒DB: 投稿プログラム（旧／新）SP/AP/PP の各Step完了日、受講開始日
クラス名 / 講師ID / アカウントURL生徒DB: 生徒情報管理ID, 表示名, クラス名(講師名), 講師ID, 【提出フォーム】アカウントURL
卒業判定 / 初回HR / フォロワー生徒DB: 要サポート生徒分析（新）生徒名, 卒業生or現役生, 卒業日, 初回HR日, 最終投稿日, 直近未投稿日数, 今日のフォロワー, 合計投稿数
講義実施日 / 出欠講師DB: クラス情報管理 → 各講師の生徒管理シート各クラスタブの1行目(回数)、4行目(実施日)、5行目以降(生徒×出欠)
各ページの使い方
ダッシュボード全体サマリ・今日の講義予定・講師カード一覧・全体集計
やること停滞・つまずき・卒業対応・未開始を優先度別にまとめた行動リスト
生徒検索生徒名・クラスから検索。ステータスでフィルタ可能
クラス一覧SP受講開始月ごとにクラスを集計・カード表示
進捗キャッチアップ状況別グループ（停滞/つまずき/進行中/達成/未開始）を折りたたみ表示
操作のコツ
  • 右上の 「全校 / 講師名を入力」 で講師を選ぶと、その講師の生徒だけに絞り込まれます。次回も自動で復元。
  • 「✕」ボタンで全校表示に戻ります。
  • 各KPIカードや生徒名はクリック可能。詳細表示や別ページへの導線になっています。
  • ヘッダー左の 「← 戻る」 ボタンで直前の画面に戻れます。
  • 講義スケジュールは講師選択中だけ表示され、生徒管理シートを自動で取得して埋まります。
