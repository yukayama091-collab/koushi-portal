# 講師ポータル

SnsClub 講師用ポータル — 担当生徒の進捗・卒業状況・講義スケジュールを一元的に把握するための内部ツール。

## 構成

- `koushi_portal.html` — 単一ファイルのフロントエンド（Chart.js + Vanilla JS）
- 生徒DB／講師DBの接続には **GAS Web App を受付窓口** として経由（非公開シートを安全に読むため）

## 起動

ローカルHTTPサーバー経由で開きます（`file://` 直開きはCORSで動きません）:

```bash
cd <このリポジトリのフォルダ>
python3 -m http.server 8765
```

ブラウザで `http://localhost:8765/koushi_portal.html` を開き、右上の **⚙️ 設定** からGAS Web AppのURLとアクセストークンを入力してください（初回のみ。以降はブラウザに保存）。

## 主な機能

- 📊 **ダッシュボード**: 今日の講義・全講師カード一覧・全体集計
- 📋 **やること**: 停滞・つまずき・卒業対応・未開始を優先度別に
- 🔍 **生徒検索**: 生徒名・状態でフィルタ
- 📚 **クラス一覧**: 開始月ごとに集計
- 📈 **進捗キャッチアップ**: 状況別に折りたたみ表示
- ❓ **ヘルプ・凡例**: マーク・色・データソースの説明

## 卒業判定ロジック

- プログラム完了（PP10完了日）あり、もしくは要サポート分析で `卒業生`＋卒業日が過去 → 卒業
- 卒業予定日: 第1回（初回HR） + 180日。例外: 第12回がそれより遅ければ第12回。
